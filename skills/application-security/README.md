# application-security

Application security skill package for threat modeling, secure coding review, authentication and authorization patterns, API hardening, data protection, and security operations guidance.

## Install

```bash
npx skills add https://github.com/rksharma-owg/Codex-Skills --skill application-security
```

## Package Contents

- `SKILL.md`
- `references/`

## Usage Notes

- Run the install command above to add this package from its current location in this repository.
- Open `SKILL.md` for the main skill behavior and prompts.
- Use the `references/` directory for deeper security implementation guidance.
