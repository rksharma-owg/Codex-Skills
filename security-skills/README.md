# Security Skills

This section contains dedicated security skill packages with their supporting files and source notes.

## Design Principle

- Curated summaries belong in `skills/`.
- Security-focused packaged skills belong in `security-skills/`.
- Source notes, licensing clues, and original supporting materials stay visible here.

## Included Packages

| Package | Source Notes |
| --- | --- |
| [application-security](./application-security/SOURCE.md) | Security package based on `oakoss/agent-skills`. |

## Handling Guidance

- Keep packaged skill contents intact unless a maintenance task explicitly requires a patch.
- Preserve author, license, and source context wherever it is available.
- Avoid blending these packaged security skills into the curated summaries in `skills/`.
